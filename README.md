# Project-ESPNow
未命名，可称为无线小纸条，无线同步触摸或者其他任意二创/开发名称
此为开源项目，适用于常见的ESP32-CYD总成主板。Kurio Reiko感激一切对于本项目的支持，包括语言鼓励、项目转载、代码修缮或者产品再分发。商业化销售是被允许的，但是如果有人在未经声明原创作者而直接照搬代码的商业化销售，那么他/她/它/They就是个傻逼。
